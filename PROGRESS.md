  5. **`android:exported` manifest error**: targetSdk bump (30 → `flutter.targetSdkVersion`, resolves to 36) crosses the Android 12 threshold requiring every intent-filtered activity to declare `android:exported` explicitly. Added `android:exported="true"` to `MainActivity` in `AndroidManifest.xml`.
  6. **JVM target mismatch**: old plugins (e.g. `flutter_native_timezone`) don't declare `compileOptions`, defaulting to Java 8, while the bumped Kotlin plugin defaults Kotlin compilation to JVM 17 — extended the same `subprojects{}` patch to force `compileOptions` (Java 17) and Kotlin `compilerOptions.jvmTarget` (17) across every plugin subproject.
